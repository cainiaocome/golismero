Base Url type¶

Base URL type.

class golismero.api.data.resource.baseurl.BaseUrl(url)¶

Bases: golismero.api.data.resource.Resource

Base URL.

Unlike the Url type, which refers to any URL, this type is strictly for root level URLs in a web server. Plugins that only run once per web server should probably receive this data type.

For example, a plugin receiving both BaseUrl and Url may get this input:

  • BaseUrl(“http://www.my_site.com/”)
  • Url(“http://www.my_site.com/”)
  • Url(“http://www.my_site.com/index.php”)
  • Url(“http://www.my_site.com/admin.php”)
  • Url(“http://www.my_site.com/login.php”)

Notice how the root level URL is sent twice, once as BaseUrl and again the more generic Url.
