class golismero.api.data.vulnerability.information_disclosure.url_disclosure.UrlDisclosure(url, **kwargs)¶

Bases: golismero.api.data.vulnerability.UrlVulnerability

URL disclosure vulnerability.

These are URLs that are accessible but not linked from the web site itself. This may indicate a poor attempt at concealing information.

For example:

  • Backup files: http://www.my_site.com/ index.php.old
  • Alternative file names: http://www.my_site.com/ index4.php
  • Remnants of deployment: http://www.my_site.com/ build.xml
  • Poorly configured servers: http://www.my_site.com/ error_log
  • Forgotten server files: http://www.my_site.com/ server-status
discovered¶

Discovered part of the URL.

>>> from golismero.api.data.resource.url import Url
>>> from golismero.api.data.vulnerability.information_disclosure.url_disclosure import UrlDisclosure
>>> url = Url('http://www.my_site.com/path/to_the/file/index.php.old')
>>> url_disclosure = UrlDisclosure(url)
>>> url_disclosure
<UrlDisclosure url='http://www.my_site.com/path/to_the/file/index.php.old'>
>>> url_disclosure.discovered
'index.php.old'
Returns:Discovered part of the URL.
Return type:str

Url suspicious¶

Suspicious URLs that could require manual attention.

class golismero.api.data.vulnerability.information_disclosure.url_suspicious.SuspiciousURL(url, substring, *argv, **argd)¶

Bases: golismero.api.data.vulnerability.UrlVulnerability

Suspicious URLs that could require manual attention.

For example:

  • http://www.my_site.com/users/ -> URL with suspicious path.
  • http://www.my_site.com/index.jsp? jsessionid =.... -> suspicious parameter
  • http://www.my_site.com/index.php?id=1 -> no suspicious info
substring¶

Matched substring within the URL that causes suspicion.

>>> from golismero.api.data.resource.url import Url
>>> from golismero.api.data.vulnerability.information_disclosure.url_suspicious import SuspiciousUrl
>>> url = Url('http://www.my_site.com/path/index.jsp?para1=1&jsessionid=901828392012')
>>> suspicious = SuspiciousUrl(url, 'jsessionid')
>>> suspicious
<SuspiciousUrl url='http://www.my_site.com/path/index.jsp?para1=1&jsessionid=901828392012'>
>>> suspicious.substring
'jsessionid'
