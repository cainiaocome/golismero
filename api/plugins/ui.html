UI plugins¶

There are the list of plugins that are responsible of generate the User Interface:

Console plugin¶

class plugins.ui.console.ConsoleUIPlugin¶

This is the console UI plugin. It provides a simple interface to work with GoLismero from the command line.

This plugin has no options.

plugins.ui.console.process_default_error_page(vuln)¶

Display default error page

plugins.ui.console.process_url(url)¶

Display URL info

plugins.ui.console.process_url_disclosure(vuln)¶

Display URL discover

plugins.ui.console.process_url_suspicious(vuln)¶

Display suspicious URL
